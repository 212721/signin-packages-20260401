# 自动签到程序打包

本仓库包含三个自动签到/续期程序：

## 程序列表

### 1. senidc-points-signin.tar.gz - 森云积分商城自动签到
- 每日自动签到获取积分
- 支持邮件通知
- 需要配置账号密码
- 打包时间: 2026-04-01

### 2. domecdn-renew.tar.gz - DomeCDN自动续期
- 自动检测并续期CDN服务
- 避免服务过期
- 需要配置CDN账号信息
- 打包时间: 2026-04-01

### 3. tg-user-keeper.tar.gz - Telegram机器人签到管理
- 管理多个Telegram机器人的自动签到
- 支持10+个不同机器人
- 包含健康检查和自动恢复功能
- 打包时间: 2026-04-01

## 使用说明

1. 下载对应的tar.gz文件
2. 解压: `tar -xzf filename.tar.gz`
3. 按照各自目录下的README.md进行配置
4. 运行程序

## 注意事项

- 所有配置文件中的敏感信息已排除
- 请解压后根据实际情况配置相应的账号密码
- 建议在测试环境先验证配置是否正确
